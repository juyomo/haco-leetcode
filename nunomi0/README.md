## 📍 알고리즘 스터디 기록 규칙


### 플랫폼 약어

| 약어 | 플랫폼 이름                  |
|:----:|:----------------------------|
| `BOJ`  | 백준 (Baekjoon Online Judge) |
| `PGS`  | 프로그래머스 (Programmers)   |
| `LTC`  | 리트코드 (LeetCode)          |
| `CFS`  | 코드포스 (Codeforces)        |
| `SEA`  | 삼성 SW Expert Academy       |
| `ETC`  | 기타 (Etc)                   |


### 커밋 메시지 규칙

```bash
git commit -m "[약어] 문제 제목.확장자" -m "문제 링크"
```

- `약어`: 문제 플랫폼 (위 표 참고)
- `문제 제목`: 문제 이름 (띄어쓰기 허용)
- `확장자`: 사용한 언어 (예: `.cpp`, `.js`, `.py` 등)
- `문제 링크`: 해당 문제 URL (두 번째 `-m`에 작성)
